# 耀州窑青釉刻花碗

**id**: artifact_001
**era**: 北宋
**category**: 瓷器
**museum**: 广州博物馆
**origin**: 陕西省铜川市黄堡镇
**summary**: 北宋瓷器代表器物：耀州窑青釉刻花碗

## description
耀州窑位于今陕西铜川黄堡镇，始烧于唐代，延续至元代，历时八百余年，在中国陶瓷史上占有重要地位。其瓷器风格影响广泛，多个窑口曾仿烧耀州窑产品。北宋时期刻花青瓷尤为精美，器型多样，釉色温润。本件青釉刻花碗仿莲花造型，釉面光洁莹润。 该器物在北宋时期具有典型代表性，反映了当时瓷器的发展水平、审美取向以及社会文化背景，在艺术史与工艺史研究中具有重要价值。
